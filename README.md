# statsboard
(Working Title)

## Installation
```
npm install
```

## Usage

Run Development Server (Has live reloading)
```
npm run dev
```

Run production server
```
npm run build && npm start
```

## Folder Stucture
- `public`: Static CSS/Images/etc.
- `pages`: Main pages
- `components`: React Components
- `pages/api`: Routes any `/api/*` requests
